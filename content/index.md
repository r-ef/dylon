#title
dylon : Effortless Code Scaffolding.

#description
Rapid project setup made easy.
Pre-built templates. One-command deployment. Open Source.

<br></br>

::code-group
  ```bash [npm]
  npm i -g dylon
  ```

  ```bash [pnpm]
  pnpm i dylon
  ```

  ```bash [yarn]
  yarn add dylon
  ```
::

::code-group
  ```bash [Terminal]
  # get list of available options
dylon --help

# scaffold a new project
dylon init
  ```
